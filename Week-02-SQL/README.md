# Week 2 Assignment

This is my Week 2 assignment submitted as part of the CEI Program for Data Engineering .

The assignment is based on an e-commerce sales database with four tables: customers, products, orders, and order_items. I created the tables, loaded the given sample data into Databricks, and solved SQL questions covering topics such as SELECT queries, filtering with WHERE, aggregate functions, GROUP BY, HAVING, joins, CASE statements, constraints, transactions, and ACID properties.

Through this assignment, I gained hands-on experience working with relational databases and writing SQL queries to analyze business data. I learned how to retrieve specific records, calculate summaries using aggregate functions, and combine data from multiple tables using joins. The analysis also helped me understand sales trends, customer orders, and product information within an e-commerce environment.

Overall, this assignment improved my understanding of SQL fundamentals and gave me practical experience in querying and analyzing data using Databricks.
